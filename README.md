Customer Segmentation & Retention Analysis using RFM and K-Means

📌 Project Overview

This project focuses on identifying high-value customers and improving retention strategies using RFM Analysis and K-Means Clustering.

Customer segmentation helps businesses:

Identify loyal customers

Detect churn risk

Personalize marketing campaigns

Improve revenue through targeted strategies

🎯 Objectives

Perform RFM (Recency, Frequency, Monetary) Analysis

Segment customers based on purchasing behavior

Apply K-Means clustering for grouping customers

Visualize customer segments

Build an interactive Power BI Dashboard for insights

🧰 Tools & Technologies Used

Python

Pandas

NumPy

Scikit-learn

Matplotlib

Power BI

📊 Methodology
1. Data Preprocessing

Handled missing values

Removed duplicates

Converted date columns

Calculated Total Purchase Value

2. RFM Analysis

Customers were evaluated using:

Recency – Last purchase date

Frequency – Number of transactions

Monetary – Total spending

3. K-Means Clustering

Used Elbow Method to determine optimal clusters

Segmented customers into different groups

📈 Customer Segments Identified

High-Value Customers

Loyal Customers

At-Risk Customers

Lost Customers

📊 Dashboard Insights

The Power BI dashboard provides:

Segment distribution

Revenue contribution by segment

Customer retention patterns

RFM score visualization
